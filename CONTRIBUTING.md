# Contributing

Thank you for your interest in contributing to Awesome Agentic Commerce.

## Guidelines

- Only **official sources** are accepted: specs, official documentation, SDKs, and official blog posts from the maintaining organizations.
- Please open an issue or pull request to suggest new links or report broken ones.

## Link checklist

Before submitting, ensure any new or changed links are valid. From the repo root you can run:

```bash
lychee README.md
```

(or use `scripts/check-links.sh` if available).
